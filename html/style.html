Style-Tag

The <style> tag in HTML helps us to modify our text, viewed in the page. This modification includes changing font size, font family, font color etc. Not only the texts but also we can change the style of a body or part of a page. Now letâ€™s look at various attributes of style and what else the tag supports.

Syntax

<tagname style="property:value">

1) The tagname includes <p>, <body>, from <h1> to <h6> etc.
2) The property is borrowed from CSS like color, font-size, font-family etc.
3) The value is also borrowed from CSS.

1) color-property
