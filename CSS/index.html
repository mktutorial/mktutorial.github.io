An inline CSS uses the style attribute of an HTML element. An internal CSS is defined in the <head> section of an HTML page, within a <style> element. To use an external style sheet, add a link to it in the <head> section of each HTML page
:<link rel="stylesheet" type="text/css" href="styles.css">
